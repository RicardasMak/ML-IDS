This code will run only on Linux OS and with root permission.
To set up environment for the code please launch bash file “dependancys.sh”.
It will automatically install python 3.9 and python libraries to run the code.
Note bash file will only run on Debian based Linux or other distribution which
supports packet manager “apt”.
Once the installation is complete “main.py” can be executed via terminal.

----------------

Dataset is included with name “log.csv”. When building dataset, it will
append new data into the “log.csv”. As well for building machine learning 
model it will use log.csv as training dataset. 

---------------

All four machine learning models are included with corresponded naming.
